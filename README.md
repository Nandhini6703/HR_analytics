# 📊 HR Analytics Dashboard
# 📌 Project Overview
This project analyzes employee data to understand workforce distribution, hiring trends, and employee demographics. The dashboard provides key insights that help HR teams make better workforce planning and management decisions.
# 🎯 Key KPIs
Total Number of Employees
Remote vs Headquarters Employees
Average Employee Age
Hiring Rate by Year
Gender Distribution
State-wise Employee Distribution
Department-wise Employee Distribution
Race-wise Employee Distribution
# 🛠 Tools Used
Power BI
Excel 
Data Cleaning & Data Visualization
# 📈 Key Insights
The organization has around 22,000 employees, indicating a large workforce.
75% employees work from headquarters, while around 25% work remotely.
The average employee age is 38.4, showing a moderately experienced workforce.
Engineering department has the highest number of employees, indicating a technology-driven organization.
The company shows diversity in gender and race distribution.
